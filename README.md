# DLG Connect - Automação Profissional para Telegram

Sistema de automação avançado para crescimento de grupos Telegram.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```

Os arquivos serão gerados na pasta `dist/`.

## Deploy

Faça upload da pasta `dist/` para seu servidor de hospedagem.
